# IPL Visualization Project

## Dataset
IPL ball-by-ball deliveries data

## Tools
Python, Pandas, Matplotlib, Seaborn, Plotly

## Key Findings
1. Virat Kohli — highest run scorer
2. Death overs score most runs
3. Caught is most common dismissal
4. Wide ball most common extra
5. Top team scored most total runs

## Charts Built — 10 charts
Bar charts, line charts, interactive Plotly dashboard
